Instruction Budget

Budget model

Repository instructions — AGENTS.md + CLAUDE.md

Provider-specific root baselines measured independently. Doctrine loads use the active provider baseline and never sum these files.

Agents — instructions per .claude/agents/*.md

Skills — instructions per SKILL.md

Doctrine loads — instructions per session before the contract

+ canonical agent instructions + named skill. Ratio is doctrine : slice budget for sessions that execute or check a slice.

Chains — cross-context gauge

Summed doctrine across every hop. Context-window boundaries isolate each hop at runtime; bucket for trend, not for alarm.

Suggestions

    Method: an instruction is a sentence outside code blocks and frontmatter; tokens are chars / 4. Dynamic content — briefs, contracts, review ledgers, tool feedback, self-instruction — is unmeasured and spends the remaining headroom. Produced by $instruction-budget.
